# PRP_Model
a simple model of prp solved by cplex, from undergraduate paper.
### 文件介绍
- uut_final.mod: PRP的模型所有软约束和硬约束一起求解
  - data_fianl.dat: 对应的数据
- uut_v3.mod: 只考虑硬约束
  - data_v3.dat: 对应的数据
